# Comandos Úteis - DOGUP Contábil

## Desenvolvimento
npm run dev          # Iniciar servidor de desenvolvimento
npm run build        # Build de produção
npm run preview      # Preview do build de produção
npm run check        # Verificar erros TypeScript
npm run sync         # Sincronizar tipos do SvelteKit

## Prisma (Banco de Dados)
npx prisma generate                  # Gerar cliente Prisma
npx prisma studio                    # Abrir interface gráfica do banco
npx prisma db push                   # Push schema para banco (criar/atualizar tabelas)
npx prisma migrate dev               # Criar/rodar migrations
npx prisma migrate deploy            # Aplicar migrations em produção

## Seed (Dados Demo)
node prisma/seed.cjs                 # Popular banco com dados de demonstração

## Node/npm
node -e "script"                     # Executar código JavaScript inline
npm install                          # Instalar dependências
npm install <package> --save        # Instalar package em produção
npm install <package> --save-dev     # Instalar package em desenvolvimento
npm uninstall <package>             # Remover package

## Git
git status                          # Ver estado do repositório
git add .                           # Adicionar todos arquivos
git commit -m "mensagem"            # Criar commit
git push                            # Enviar para remote
git pull                            # Baixar do remote
git branch                          # Listar branches
git checkout <branch>               # Trocar de branch
git merge <branch>                   # Mesclar branch

## Variáveis de Ambiente

SESSION_SECRET                      # Chave secreta para sessões
